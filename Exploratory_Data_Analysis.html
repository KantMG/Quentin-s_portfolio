Exploratory Data Analysis

OBJECTIVE

Development of an advanced user-friendly visualization and analysis tool for profiling dataframes.

Libraries required

  • Data Analysis Libraries:
    NumPy - Pandas - SciPy
  • Data Visualization Tools:
    Plotly - Matplotlib
  • Machine learning:
    scikit-learn
  • Tableau:
    Dash

Interface properties

The interface presents two tabs. One tab is dedicated to the profiling of the dataframe. The second is made to have a deep analysis of the data.

Tab 1 : Profiling

  • All the basic information regarding the properties of the dataframe.
  • Target/Features correlation.
    Numerical feature: Correlation Matrix
    Categorical feature: Crosstab
  • Hypothesis Testing Methods:
    If the conditions are fulfilled: t-test - ANOVA - chi2
    If not: Welch's ANOVA - Kruskal-Wallis test - Mann-Whitney U Test

Tab 2 : Analysis

The interface offers a wide range of plot management capabilities through various Dash callback functions.
  • Data filtration.
  • Creation of functions.
  • Smoothing with the Savitzky-Golay filter.
  • Machine learning regressions.
  • Dynamic subplot modification.
